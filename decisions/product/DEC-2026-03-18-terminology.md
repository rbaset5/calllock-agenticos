# Terminology: App not Dashboard

## Context
The CallLock web application was being referred to as "the dashboard" in agent conversations and documentation.

## Decision
Always use "CallLock App" or "CallLock Web App" — never "dashboard." It's a full application that contractors use to review calls and manage their business, not a read-only metrics view.

## Consequences
- All agent instructions, documentation, and conversation must use "CallLock App."
- Agents that use "dashboard" are providing incorrect context to downstream tools.

# Seed Data / Demo Mode Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add `?demo=true` URL param that serves 30+ static seed calls instead of fetching from Supabase, with a demo banner and realtime disabled.

**Architecture:** A static `Call[]` array in `src/data/seed-calls.ts` is returned directly from `page.tsx` when `searchParams.demo === "true"`. The `Mail` component accepts an `isDemo` prop that suppresses the Supabase realtime subscription and shows a banner.

**Tech Stack:** Next.js 16 App Router (async searchParams), TypeScript, Vitest

```ts
    for (const call of SEED_CALLS) {
      expect(call.transcript.length).toBeGreaterThanOrEqual(2)

      expect(typeof call.id).toBe("string")

      expect(typeof call.createdAt).toBe("string")

  })

    const booked = SEED_CALLS.filter((c) => c.appointmentBooked)

  it("has at least one safety emergency", () => {
```

**Step 1: Write the failing test**

**Step 2: Make the change**

In `src/hooks/use-realtime-calls.ts`, add `isDemo?: boolean` as a third parameter and guard the subscription effect:

```ts
export function useRealtimeCalls(
  initialCalls: Call[],
  readIds: Set<string>,
  isDemo = false
) {
  const [calls, setCalls] = useState<Call[]>(initialCalls)
  const readIdsRef = useRef(readIds)
  readIdsRef.current = readIds

  useEffect(() => {
    setCalls(initialCalls)
  }, [initialCalls])

  useEffect(() => {
    if (isDemo) return   // ← skip subscription in demo mode

    const supabase = createBrowserClient()
    const channel = supabase
      .channel("call_sessions_changes")
      .on(
        "postgres_changes",
        { event: "INSERT", schema: "public", table: "call_sessions" },
        (payload) => {
          const row = payload.new as CallSessionRow
          const call = transformCallSession(row, readIdsRef.current)
          setCalls((prev) => [call, ...prev])
        }
      )
      .subscribe()

    return () => { supabase.removeChannel(channel) }
  }, [isDemo])

  useEffect(() => {
    setCalls((prev) => prev.map((c) => ({ ...c, read: readIds.has(c.id) })))
  }, [readIds])

  return calls
}
```

### Task 4: Update `page.tsx` to branch on `?demo=true`

**Files:**
- Modify: `src/app/page.tsx`

**Step 1: Add searchParams and demo branch**

In Next.js 16, `searchParams` is a `Promise` — it must be awaited:

```ts
import { createServerClient } from "@/lib/supabase"
import { transformCallSession } from "@/lib/transforms"
import { Mail } from "@/components/mail/mail"
import { SEED_CALLS } from "@/data/seed-calls"
import type { Call, CallSessionRow } from "@/types/call"

export const dynamic = "force-dynamic"

export default async function CallsPage({
  searchParams,
}: {
  searchParams: Promise<{ demo?: string }>
}) {
  const params = await searchParams

  // Demo mode — skip Supabase entirely
  if (params.demo === "true") {
    return (
      <div className="flex h-full flex-col">
        <Mail initialCalls={SEED_CALLS} isDemo={true} />
      </div>
    )
  }

  let calls: Call[] = []

  try {
    const supabase = createServerClient()
    const { data, error } = await supabase
      .from("call_sessions")
      .select("call_id, conversation_state, created_at, synced_to_dashboard")
      .order("created_at", { ascending: false })
      .limit(100)

    if (!error && data) {
      const rows = data as CallSessionRow[]
      const emptyReadIds = new Set<string>()
      calls = rows.map((row) => transformCallSession(row, emptyReadIds))
    }
  } catch {
    // Supabase unreachable — render empty state
  }

  return (
    <div className="flex h-full flex-col">
      <Mail initialCalls={calls} />
    </div>
  )
}
```
